== Description ==

Style My Tweets adds and easy to use options page for styling the Twitter widget that comes with Jetpack by WordPress.com.  There is a good chance this plugin will work with other twitter widgets as well.

For more information on Themekit, check out [themekitwp.com](http://themekitwp.com/).

Features include:

* Options to style the widget container
* Options to style the widget header 
* Options to style the individual tweets


== Installation ==

1. Install ThemeKit either via the WordPress.org plugin directory, or by uploading the files to your server (if you already have Themekit installed you may only need to upgrade to the latest version)
2. Install the Style My Tweet plugin is the same what you installed Themekit
2. Activate both plugins.
3. That's it.  You're ready to go!
4. Enjoy your awesome new options.


== Frequently Asked Questions ==

= Why do I need Themekit for this plugin to work? =

Themekit is required because it is a great way to build option pages for WordPress.
